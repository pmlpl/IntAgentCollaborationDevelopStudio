# Phase 1.5 终端 UX Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 实现美观 Textual 指挥舱 + 多终端 Agent spawn，用户运行 `studio` 即可完成开公司向导、下任务、看进度，并自动弹出主管/下属工作终端。

**Architecture:** Textual App 为 CEO 控制台；`core/terminal/spawner.py` 在 Windows 新窗口启动 Agent CLI；Dispatcher 扩展主管 JSON 拆解与 Worker 派发；`runtime/state.json` 驱动进度面板。

**Tech Stack:** Textual, Rich, questionary, Python 3.11+, 现有 agents/ + dispatcher

**Spec:** [2026-06-07-phase1.5-terminal-ux-design.md](../specs/2026-06-07-phase1.5-terminal-ux-design.md)
